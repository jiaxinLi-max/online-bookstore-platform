# Online Bookstore Platform

An online bookstore platform that supports book browsing, ordering, and user management.
The system is built with a Java-based backend and a Vue frontend, following a typical
frontend-backend separation architecture.

## Tech Stack
**Backend**
- Java
- Spring Boot
- RESTful API
- MySQL
- JWT-based authentication

**Frontend**
- Vue
- Vue Router
- Axios
- Element UI

## Features
- User registration and login
- Book listing and search
- Shopping cart management
- Order creation and order history
- Admin book management (add / update / remove books)
